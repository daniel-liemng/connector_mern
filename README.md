## About The Project
A web application platform to widen the networking. Logged in user can create profile, add posts and add comments. Guest user can view the other users' profile.

## Tech Stack
  * **MERN: MongoDB, Express, ReactJS, NodeJS
  * **Material-UI
  * **Mongoose
  * **JSON Web Token
